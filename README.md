# 招生代理管理系统

《招生代理管理系统》是一款管理代理招生的软件,各自代理录入各自的信息,管理员可以查看全部数据，便于管理学员信息。 系统包含：基本信息、学员管理、统计汇总 1、基本信息：单位信息、业务提成、用户分配 2、学员管理：学员信息 3、统计汇总：学员查询、提成查询 

# 如何运行本系统?

本系统基于宏达数据库信息管理开发平台开发，下载后，双击文件夹中"Hadp.exe"文件即可完整运行对应的信息管理系统。
